# Syntax-Saga

### Python Venv

```
cd .\Backend\
syntax-saga\Scripts\activate
cd fastapi_service
uvicorn deepseek_microservice:app --reload --port 8001

cd .\Backend\
syntax-saga\Scripts\activate
python app.py
```

### Run Backend

```
cd .\Backend\
node server.js
```

### Run Frontend

```
cd .\Frontend\
npm run dev
```
